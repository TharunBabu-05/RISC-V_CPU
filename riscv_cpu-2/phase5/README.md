# Phase 5 Workspace

This directory collects the implementation-facing artifacts for the Phase 5 handoff.

## Areas

- compliance: riscv-tests and riscv-arch-test execution profiles
- formal: SymbiYosys jobs for control, hazard, and CSR checks
- synthesis: timing and clock/reset constraint templates

## Status

- The repo already contains the RTL needed for the phase-5 signoff paths.
- These files provide the first implementation scaffold for the remaining toolchain work.
